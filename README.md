# Bomberman 
 
Bomberman clone written in python using pygame. 
 
Game allow to play against two types of AI (Dijkstra Algorithm and Depth-first search) 
 
![](images/git1.png)
 
## Controls 
WSDA - move 
Space - plant bomb 
 
## Run 
`` 
python3 meny.py 
`` 
 
For now game is not compatible with python2
